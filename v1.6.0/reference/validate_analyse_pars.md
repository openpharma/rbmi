# Validate analysis results

Validates analysis results generated by
[`analyse()`](https://openpharma.github.io/rbmi/reference/analyse.md).

## Usage

``` r
validate_analyse_pars(results, pars)
```

## Arguments

- results:

  A list of results generated by the analysis `fun` used in
  [`analyse()`](https://openpharma.github.io/rbmi/reference/analyse.md).

- pars:

  A list of expected parameters in each of the analysis. lists i.e.
  `c("est", "se", "df")`.
