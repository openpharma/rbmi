# Extract imputed datasets

Extracts the imputed datasets contained within an `imputations` object
generated by
[`impute()`](https://openpharma.github.io/rbmi/reference/impute.md).

## Usage

``` r
extract_imputed_dfs(
  imputations,
  index = seq_along(imputations$imputations),
  delta = NULL,
  idmap = FALSE
)
```

## Arguments

- imputations:

  An `imputations` object as created by
  [`impute()`](https://openpharma.github.io/rbmi/reference/impute.md).

- index:

  The indexes of the imputed datasets to return. By default, all
  datasets within the `imputations` object will be returned.

- delta:

  A `data.frame` containing the delta transformation to be applied to
  the imputed dataset. See
  [`analyse()`](https://openpharma.github.io/rbmi/reference/analyse.md)
  for details on the format and specification of this `data.frame`.

- idmap:

  Logical. The subject IDs in the imputed `data.frame`'s are replaced
  with new IDs to ensure they are unique. Setting this argument to
  `TRUE` attaches an attribute, called `idmap`, to the returned
  `data.frame`'s that will provide a map from the new subject IDs to the
  old subject IDs.

## Value

A list of data.frames equal in length to the `index` argument.

## See also

[`delta_template()`](https://openpharma.github.io/rbmi/reference/delta_template.md)
for creating delta data.frames.

[`analyse()`](https://openpharma.github.io/rbmi/reference/analyse.md).

## Examples

``` r
if (FALSE) { # \dontrun{
extract_imputed_dfs(imputeObj)
extract_imputed_dfs(imputeObj, c(1:3))
} # }
```
