# Create and validate a `sample_list` object

Given a list of `sample_single` objects generate by
[`sample_single()`](https://openpharma.github.io/rbmi/reference/sample_single.md),
creates a `sample_list` objects and validate it.

## Usage

``` r
sample_list(...)
```

## Arguments

- ...:

  A list of `sample_single` objects.
