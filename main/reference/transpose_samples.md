# Transpose samples

Transposes samples generated by
[`draws()`](https://insightsengineering.github.io/rbmi/reference/draws.md)
so that they are grouped by `subjid` instead of by sample number.

## Usage

``` r
transpose_samples(samples)
```

## Arguments

- samples:

  A list of samples generated by
  [`draws()`](https://insightsengineering.github.io/rbmi/reference/draws.md).
