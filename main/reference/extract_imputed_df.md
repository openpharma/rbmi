# Extract imputed dataset

Takes an imputation object as generated by
[`imputation_df()`](https://openpharma.github.io/rbmi/reference/imputation_df.md)
and uses this to extract a completed dataset from a `longdata` object as
created by
[`longDataConstructor()`](https://openpharma.github.io/rbmi/reference/longDataConstructor.md).
Also applies a delta transformation if a `data.frame` is provided to the
`delta` argument. See
[`analyse()`](https://openpharma.github.io/rbmi/reference/analyse.md)
for details on the structure of this `data.frame`.

Subject IDs in the returned `data.frame` are scrambled i.e. are not the
original values.

## Usage

``` r
extract_imputed_df(imputation, ld, delta = NULL, idmap = FALSE)
```

## Arguments

- imputation:

  An imputation object as generated by
  [`imputation_df()`](https://openpharma.github.io/rbmi/reference/imputation_df.md).

- ld:

  A `longdata` object as generated by
  [`longDataConstructor()`](https://openpharma.github.io/rbmi/reference/longDataConstructor.md).

- delta:

  Either `NULL` or a `data.frame`. Is used to offset outcome values in
  the imputed dataset.

- idmap:

  Logical. If `TRUE` an attribute called `idmap` is attached to the
  return object which contains a `list` that maps the old subject ids
  the new subject ids.

## Value

A `data.frame`.
